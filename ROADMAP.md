# MultitoolApp Roadmap

## Version Planning

### v1.0.0 - MVP (Complete)

**Core Features**
- Calculator (basic operations)
- Unit Converter (4 categories: length, weight, volume, temperature)
- Dark/Light theme system
- Bottom tab navigation
- Modern UI/UX design
- AsyncStorage theme persistence
- TypeScript support

---

### v1.1.0 - QR Code Feature (Complete)

**Added Features**
- QR Code Generator
- Quick templates (Web, Email, Phone, WhatsApp, WiFi)
- Size options (150px, 200px, 250px)
- Share functionality
- Theme-aware QR colors

---

### v1.2.0 - Enhanced Features (Planned)

**Calculator Improvements**
- Calculation history
- Value selection from history
- Parentheses support
- Scientific calculator mode
- Copy/Paste support

**Unit Converter Improvements**
- Favorite conversions
- Recently used units
- Additional categories:
  - Area
  - Speed
  - Time
  - Data size
  - Pressure
  - Energy

**UX Improvements**
- Haptic feedback
- Animations and transitions
- Gesture support (swipe, long-press)

---

### v1.3.0 - Additional Tools (Planned)

**New Tools**
- QR code scanner
- Tip calculator
- Percentage calculator
- Discount calculator
- Loan/interest calculator
- BMI calculator

**Features**
- Widget support (iOS & Android)
- Siri/Google Assistant shortcuts
- Apple Watch support
- Tablet optimization
